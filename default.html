Pomodoro instructions

  1. Choose a task to be accomplished
  2. Start the Pomodoro timer.
  3. Work on the task until the Pomodoro rings.
  4. Take a short break.
  5. Every 4 Pomodoros take a long break.

About

This small web app was created to demonstrate new features of HTML5 and CSS3, but grew to it's current state where I (and some other people) use it whenever I need my work done.
